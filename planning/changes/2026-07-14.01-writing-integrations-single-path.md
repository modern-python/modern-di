# Change: Writing-integrations doc teaches only the integration kit

**Lane:** lightweight — one doc file, no code, no public-API change, deletion-heavy.

## Goal

The integration-kit shipping PR
([`2026-07-13.02-integration-kit.md`](2026-07-13.02-integration-kit.md))
updated `docs/integrations/writing-integrations.md` to teach the kit
primitives *alongside* the hand-rolled reference implementation, and said:
"Once the last non-outlier adapter converts, delete the old hand-rolled code
blocks (the isinstance-loop snippet, the `_FromDI`/`_parse_inject_params`
walkthrough) rather than leaving two parallel paths." All 12 framework
adapters have now converted (starlette 2.2.0 → aiogram 2.2.0), so this
removes the parallel path and leaves the kit as the single documented way.

## Approach

Three edits to `docs/integrations/writing-integrations.md`, all removing the
"here's the hand-rolled way → now use the kit instead" duality:

- **Contract point 4 (child-container builder):** replace the hand-rolled
  isinstance-loop `build_di_container` with the
  `integrations.classify_connection` version, and fold the separate "Deriving
  scope and context with the integration kit" section (single-provider `bind`,
  no-connection direct call) into point 4 as a follow-on paragraph. The
  `Container` context-manager note is kept.
- **Contract point 5 (`FromDI` + resolver):** replace the hand-rolled
  `Dependency` dataclass (field `dependency`, body
  `resolve_dependency(self.dependency)`) and the "Hold a `Marker` instead"
  transitional block with a single `Marker`-based `Dependency` taught
  directly.
- **"How the existing integrations realize the contract" table:** update the
  `FromDI` bridge row so each adapter's `Dependency`/`_Dependency` shows it
  wraps `integrations.Marker`, and Typer's cell reads inert `Marker`
  (`integrations.from_di`) instead of the old `_FromDI`.

The decorator-path section already led with the kit
(`integrations.from_di`/`parse_markers`/`resolve_markers`/`is_injected`/
`mark_injected`) and needed no change.

## Files

- `docs/integrations/writing-integrations.md` — collapse the two parallel
  paths (points 4 and 5) to the kit only; refresh the contract table row.
